## Website 1: https://example.com

### 1. What HTML tags are used on the page?

The page uses the following HTML tags:

* `<html>`
* `<head>`
* `<title>`
* `<meta>`
* `<style>`
* `<body>`
* `<div>`
* `<h1>`
* `<p>`
* `<a>`

**Example Domain**

### 3. How many headings are there?

## Website 2: https://developer.mozilla.org

### 1. Find the navigation menu - what tag is it wrapped in?

The navigation menu is wrapped in the **`<nav>`** element.

### 2. How is the search bar structured?

The search bar consists of:

* A `<form>` element.
* An `<input type="search">` field.
* A search button (`<button>`).
* Supporting labels and icons for accessibility.

### 3. What happens when you hover over links?

When hovering over links:

* The text color changes.
* The cursor changes to a pointer.
* Some links display an underline or background highlight.
* A smooth CSS transition is applied.
