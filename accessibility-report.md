# Accessibility Report

## Accessibility Audit for `index.html`

### Issues Found

1. The image needed descriptive alternative text.
2. The page required a proper heading hierarchy.
3. Link text needed to be descriptive.
4. The HTML document required the `lang` attribute.
5. All form inputs needed associated labels.

### How I Fixed Them

* Added descriptive `alt` text to the image.
* Used proper heading order (`<h1>` followed by `<h2>`).
* Replaced generic link text with descriptive text such as **"Visit GitHub"**.
* Added `lang="en"` to the `<html>` tag.
* Ensured all form inputs had matching `<label>` elements.

### Final Lighthouse Accessibility Score

**12/13 (Passed)**
